# weather-requests-python

A simple Python project to practice HTTP requests and working with weather APIs using the `requests` library.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/NaipanAbdinegara-git/weather-requests-python.git

# Run the script
python weather_requests.py
```

## 📁 Project Structure

```
.
├── weather.csv
├── weather_requests.py
└── weather_requests_v2.py
```

## 📌 Notes

This project is mainly for learning purposes, focusing on:

* Making HTTP requests with Python
* Handling JSON responses
* Working with simple weather data
